## ğŸ¨ Customization

### Update Personal Information

Edit the following files to update your information:

- **Hero Section**: `src/components/Hero.js`
- **About Section**: `src/components/About.js`
- **Contact Section**: `src/components/Contact.js`
- **Footer**: `src/components/Footer.js`

### Update Social Links

Replace placeholder links in:
- `src/components/Contact.js`
- `src/components/Footer.js`
- `src/components/FloatingButtons.js`

Update the WhatsApp number and other social media URLs.

### Update Projects

Edit `src/components/Projects.js` to add/modify project details.

### Update Pricing

Modify pricing plans in `src/components/Pricing.js`.

### Add Resume

Replace `public/resume.pdf` with your actual resume file.
